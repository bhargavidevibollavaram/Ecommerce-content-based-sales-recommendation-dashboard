E-commerce Content-Based Sales Recommendation Dashboard

This project showcases a fully interactive Excel Dashboard built to analyze e-commerce customer behavior, product performance, and sales patterns.
It uses Power Query, Pivot Tables, and Slicers to create dynamic insights that refresh automatically when filters are applied.

The dashboard is ideal for Data Analyst portfolios, helping demonstrate skills in Excel reporting, data cleaning, and visualization.
